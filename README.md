# Image PPT Generator
## 由来
试过很多AI和Skill做PPT Slides，但感觉效果都不够好，效果最惊艳的还是用GPT-Image-2生成图片，然后直接拼接成PPT，但是拆分脚本，每页生成图片，调整图片，拼接，压缩size，整个过程还是挺繁琐的，所以我就做了Image PPT Generator把这一套流程自动化了。

## 简介
基于AI的演讲PPT生成工具。从演讲底稿一键生成专业PPT，支持Word文档和文本输入，通过AI自动生成脚本拆分、设计稿和PPT图片，自动化拼接成可用PPT。

## 功能特点

- **多种输入方式**：支持上传Word文档、XMind文件，或直接粘贴文本
- **AI脚本拆分**：智能分析底稿内容，自动拆分为适合PPT展示的脚本
- **AI设计稿生成**：自动生成每页设计思路和全局视觉风格
- **AI图片生成**：一键生成PPT图片，支持局部调整
- **隐私优先**：所有数据存储在本地浏览器，无需上传到任何服务器
- **跨平台**：支持纯静态模式和代理模式运行

## 快速开始

### 方式一：纯静态模式（推荐）

直接双击 `index.html` 在浏览器中打开即可使用。

### 方式二：代理模式

如果遇到跨域限制问题，可以启动代理服务器：

```bash
node server.js
```

然后在浏览器访问 `http://localhost:3000`

## 配置API

首次使用需要在界面右上角配置AI API：

1. 点击 **LLM设置** 按钮
2. 配置文本API（如GPT-4、Claude等OpenAI兼容API）
3. 配置图片API（如DALL-E、GPT-Image等）

## 使用流程

1. **步骤1 - 演讲底稿**：上传文件或粘贴演讲内容，添加人物参考图（可选）
2. **步骤2 - 脚本拆分**：AI自动拆分脚本，可拖拽调整顺序
3. **步骤3 - 设计稿**：生成全局设计语言和每页设计思路
4. **步骤4 - 图片生成**：批量生成PPT图片，支持局部调整
5. **步骤5 - 导出下载**：导出带/无备注的PPTX文件

## 技术栈

- 原生 JavaScript（无框架）
- Tailwind CSS
- PptxGenJS（PPT生成）
- mammoth.js（Word文档解析）
- JSZip（XMind解析）

## 文件结构

```

├── server.js     # 代理服务器
├── styles/       # 样式文件
└── README.md     # 本文件
```

## 注意事项

- 图片数据存储在浏览器IndexedDB中，清除浏览器数据会丢失
- 建议定期导出PPT文件备份
- 图片生成速度取决于所选AI API的性能
